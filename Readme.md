# The role of type and token frequency in moral evaluation and prevalence judgements — sequential version

This repository contains the stimulus generation scripts, analysis scripts, and data for:

Endress, A.D. (in press). *[Title]*. Psychological Science. [DOI/link]

## Repository structure

### `code`: RMarkdown stimulus generation and analysis scripts
- `social_learning_type_token_sequential.Rmd`: Stimulus generation script
- `social_learning_type_token_sequential_results.Rmd`: Analysis script
- `raw_materials`: Stimuli used for generating the experiment  
  *(faces from Bainbridge et al., 2013, are not shared here)*  
  - `placeNames.csv`: City names  
  - `scenarios.tab`: Vignettes  
  - `scenarios.xlsx`: Vignettes  
  - `sd3.csv`: Short Dark Triad questionnaire (testable.org format)

### `data`: Anonymized data from the experiments
- `data/159771_sequential_testable/results`: Main experiment  
- `data/994749_sequential_1/results`: Pilot experiment

### `trial_files`
Trial files for testable.org. Generated by `social_learning_type_token_sequential.Rmd` (output not included here).

### `lists`
Stimulus lists for testable.org. Generated by `social_learning_type_token_sequential.Rmd` (output not included here).
